# Day 01 – Basic Logic Gates Using Verilog HDL

## Objective

To understand the fundamentals of Verilog HDL by implementing and simulating basic combinational logic gates.

## Gates Implemented

* AND Gate
* OR Gate
* NOT Gate
* XOR Gate
* NAND Gate
* NOR Gate
* XNOR Gate

## Tools Used

* Xilinx Vivado 2023
* Verilog HDL

## Concepts Learned

* Module declaration
* Input and output ports
* Continuous assignment using assign
* Testbench creation
* Simulation and waveform analysis

## Verification

Each gate was verified using a dedicated testbench. The simulation waveforms matched the expected truth tables.

## Key Learning Outcomes

* Understanding of combinational logic design
* Writing synthesizable Verilog code
* Creating stimulus using testbenches
* Viewing and debugging simulation waveforms

## Future Scope

The concepts learned in this project will be used for implementing arithmetic circuits, finite state machines (FSMs), communication protocols, and processor design.
