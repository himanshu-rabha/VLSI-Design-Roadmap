## Day 03 – Multiplexers (2:1 and 4:1 MUX)

Designed and verified 2:1 and 4:1 Multiplexers using Verilog HDL. Learned the concept of data selection, select lines, and hierarchical design methodology. Implemented a 4:1 Multiplexer using three 2:1 Multiplexer modules and verified functionality through simulation and waveform analysis in Vivado.

### Concepts Learned
- Multiplexer (MUX) fundamentals
- Select line operation
- Truth table analysis
- Conditional operator in Verilog
- Module instantiation
- Hierarchical RTL design
- Testbench development
- Functional verification and waveform debugging

### Tools Used
- Verilog HDL
- Xilinx Vivado 2023

### Projects Completed
1. 2:1 Multiplexer
2. 4:1 Multiplexer using 2:1 Multiplexers
