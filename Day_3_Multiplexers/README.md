## Overview

Designed and verified 2:1, 4:1, and 8:1 Multiplexers using Verilog HDL. Learned the fundamentals of data selection, select-line control, hierarchical design methodology, module instantiation, and functional verification through simulation and waveform analysis.

## Projects Completed

### 1. 2:1 Multiplexer
- Implemented a basic 2:1 MUX using the conditional operator.
- Verified functionality using a Verilog testbench and simulation waveforms.

### 2. 4:1 Multiplexer
- Designed a 4:1 MUX using three 2:1 MUX modules.
- Applied hierarchical design concepts and module reusability.
- Verified all select-line combinations through simulation.

### 3. 8:1 Multiplexer
- Designed an 8:1 MUX using two 4:1 MUXes and one 2:1 MUX.
- Learned scalable digital design techniques using hierarchical architecture.
- Developed comprehensive testbenches and verified functionality using waveform analysis.
