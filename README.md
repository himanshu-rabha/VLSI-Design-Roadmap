# VLSI Design and Verification Roadmap

## Overview

This repository documents my journey in learning VLSI Design and Verification as part of my M.Tech in Semiconductor Technology.

The objective of this roadmap is to build a strong foundation in Digital Design, RTL Design, Verification Methodologies, Communication Protocols, AMBA Bus Architectures, and Processor Design using industry-relevant tools and workflows.

## Technologies and Tools

* Verilog HDL
* SystemVerilog
* UVM (Planned)
* Xilinx Vivado 2023
* FSM Design
* RTL Design
* Functional Verification
* Git and GitHub

### Phase 2: Communication Protocols

* UART Protocol

### Phase 3: AMBA Protocols

* APB
* AHB
* AXI

### Phase 4: Advanced RTL Projects

* DMA Controller

* Traffic Light Controller

* RISC-V Processor Design
